# Prompt Router

## Ticket types → Default agent
- Business rules unclear → PM/BA
- Cross-cutting design/refactor → Architect
- Feature/bug fix → Developer
- Tests/quality gates → QA/QE
- Auth, exposure, secrets → Security
- CI/CD, deploy, runbooks → DevOps/SRE
- Docs/onboarding → Tech Writer

## Workflows
- Existing project: prompts/workflows/existing_project_*.md
- New project: prompts/workflows/new_project_*.md
