# New Project — 02 Architecture & Technology Selection (Adaptive)

ROLE: Architect + Security + DevOps/SRE

## Inputs
- Business brief + requirements

## Tasks
1) Propose 2–3 viable stacks (not just one), considering:
   - team skills
   - time-to-market
   - cost and operations
   - compliance needs
2) Pick a recommended stack and record decisions as ADRs.
3) Define architecture, contracts, observability, security baseline.

- ai_docs/docs/ARCHITECTURE.md
- ai_docs/docs/adr/*
- ai_docs/docs/SECURITY.md
- ai_docs/docs/OBSERVABILITY.md
